Administración de Servidores LINUX


Al completar este curso de Linux, podrás aprender a operar y administrar sistemas GNU/Linux desde los primeros pasos, configuraciones y variantes, hasta tópicos avanzados que te permitirán lograr la mayor eficiencia y productividad, tanto en servidores y ambientes de producción! Conviértete en un Administrador Linux.

Examina los eventos del sistema. Aprende la seguridad fundamental en la gestión de usuarios. Delega privilegios. instala e implementa servidores de producción como Correo Electrónico, Web, Base de Datos, Archivos, DNS, FTP, NTP, Virtualización entre otros servicios, Habilita funcionalidades y módulos del kernel.

Lo que aprenderas

✔ Instalar Linux Debian y Ubuntu desde cero

✔ Configuración de servidores Linux Debian y Ubuntu

✔ Trabajar con la terminal de Linux y sus comandos desde los más básicos hasta los avanzados

✔ LVM Logical Volume Manager

✔ LibreOffice Writter

✔ Cloud Computing con Linux

✔ Manejar permisos para carpetas y Archivos Linux

✔ Instalar paquetes en Linux

✔ Redes en Linux DNS, DHCP y más

✔ Docker y teória sobre contenedores con Linux

✔ Servidores Web con Wordpress

✔ Seguridad para Desktops y Servidores Linux
